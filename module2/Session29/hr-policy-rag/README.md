# HR Policy RAG Assistant

This project is an AI-powered HR Policy Assistant that uses a Retrieval-Augmented Generation (RAG) framework (built with Python, ChromaDB, and the Google Gemini API) to ground its answers accurately in company policy documents. 

To run it, install dependencies with `pip install -r requirements.txt`, configure your `GEMINI_API_KEY` in a `.env` file, and execute `python hr_policy_rag.py`.
