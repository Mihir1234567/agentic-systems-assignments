# HR Policy RAG Assistant

A RAG-powered chatbot that answers HR policy questions using company-specific documents.
Built with Python, ChromaDB, and Google Gemini API.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Set your `GEMINI_API_KEY` in a `.env` file or environment variable.
3. Run the script: `python hr_policy_rag.py`
